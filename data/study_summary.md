# study_summary.csv — описание данных исследования «Мир эмоций»

Сводная выгрузка результатов исследования распознавания базовых эмоций у школьников с расстройством аутистического спектра. Файл соответствует формату экспорта приложения `exportAllProfilesSummaryCsv`: одна строка — один участник, агрегированные показатели диагностики до и после формирующего этапа. Детальные пооответные данные вынесены в `study_detailed.csv`.

## Что отражает файл

В исследовании участвовали **27 школьников с РАС**, распределённых на две группы:

- **Экспериментальная группа (ЭГ), 14 человек** — проходит полный протокол: первичная диагностика → цикл занятий с приложением (три модуля) → повторная диагностика;
- **Контрольная группа (КГ), 13 человек** — проходит первичную и повторную диагностику в том же временном интервале, без занятий с приложением (обучение по адаптированной образовательной программе).

Диагностика построена на **двух методиках**, по **6 заданий в каждой** (по одному на каждую базовую эмоцию Экмана: радость, грусть, гнев, страх, удивление, отвращение):

1. **«Эмоциональные лица»** (методика Н. Я. Семаго) — распознавание эмоции по мимической экспрессии. Базовая (перцептивная) методика; именно на ней строятся основные показатели точности и динамики.
2. **«Понимание эмоций в контексте социальных ситуаций»** (по подходу социальных историй К. Грей) — определение эмоции персонажа по описанию ситуации. Контекстная методика, отражающая более сложный уровень навыка.

Каждое задание имеет 4 варианта ответа, 1 правильный. Результат по эмоции фиксируется бинарно: 1 балл при правильном ответе, 0 при неправильном. Суммарный балл по методике — от 0 до 6.

Файл агрегирует **результаты диагностики** (pre/post). Обучающие метрики занятий (прогресс по модулям, адаптивная сложность, активность по дням) в этот формат не выгружаются — они доступны в приложении на экране `/analytics`.

Одна строка = один участник. Всего **41 колонка**. Базовые `pre_accuracy` / `post_accuracy` относятся к методике «Эмоциональные лица».

## Как читать одну строку

**Артём** (экспериментальная группа):

- «Эмоциональные лица»: **2/6 → 4/6** (33% → 67%). На входе узнавал радость и гнев; после занятий добавились страх и удивление.
- «Социальные истории»: 0/6 → 0/6 — контекстный компонент остался недоступен (типично: социальное понимание формируется медленнее перцептивного).
- Время реакции: **5037 → 3563 мс** — заметное ускорение.

**Михаил** (контрольная группа):

- «Эмоциональные лица»: 1/6 → 1/6 — без изменений (узнаёт только радость).
- Время реакции: 4372 → 4287 мс — практически без сдвига.

Сопоставление двух строк иллюстрирует основной замысел исследования: динамика в экспериментальной группе сопровождается занятиями с приложением, а контрольная группа за тот же период остаётся стабильной.

В экспериментальной группе положительная динамика отмечена по всем шести эмоциям, наиболее выраженно — по радости, грусти и гневу (+3 ребёнка по каждой). В контрольной группе изменение коснулось только радости (+1).

- **Радость и грусть** доступнее остальных эмоций уже на старте — они имеют наиболее контрастную и узнаваемую мимику.
- **Страх и удивление** систематически смешиваются между собой (сходная конфигурация бровей и рта), поэтому распознаются хуже и медленнее всего поддаются обучению.
- **Отвращение** — самая трудная эмоция: редка в социальном опыте детей и почти не распознаётся даже после занятий.

### По методикам

Перцептивный компонент (лица) развивается заметно сильнее контекстного (истории): суммарный балл по «Эмоциональным лицам» в экспериментальной группе вырос с 27 до 41 (из 84 возможных), по «Социальным историям» — с 8 до 13. Понимание эмоции в ситуации требует не только опознания мимики, но и связывания события, состояния персонажа и его реакции, что для школьников с РАС объективно сложнее.

## Статистический анализ

Используются непараметрические критерии (малый объём выборки, порядковый характер баллов).

1. **Внутригрупповая динамика — критерий знаковых рангов Вилкоксона** (`post − pre` по суммарному баллу):
   - ЭГ, «Эмоциональные лица»: 9 положительных сдвигов, 0 отрицательных, 5 без изменений, **T = 0, p = 0.006** (значимая динамика);
   - ЭГ, «Социальные истории»: 4 / 0 / 10, **p = 0.059** (положительная тенденция);
   - КГ по обеим методикам: **p = 1.000** (значимых изменений нет).
2. **Межгрупповое сравнение приростов Δ — U-критерий Манна–Уитни**:
   - «Эмоциональные лица»: **U = 37.5, p = 0.003** (сдвиги ЭГ значимо выше);
   - «Социальные истории»: **U = 60.0, p = 0.029** (сдвиги ЭГ значимо выше);
   - исходный уровень ЭГ vs КГ: различий нет (группы сопоставимы).
3. **Изменения по отдельным эмоциям — точный критерий Мак-Немара** (по парам бинарных наблюдений): все переходы в экспериментальной группе направлены в сторону улучшения (0 → 1, обратных нет), но из-за малых частот значимости по отдельным эмоциям не достигается (p = 0.25–1.00).
4. **Время реакции** (вспомогательный показатель): сокращение в экспериментальной группе значимо (Вилкоксон, **p < 0.001**), в контрольной — нет; межгрупповое различие приростов RT значимо.
5. **Размер эффекта** для критерия Вилкоксона — `r = Z / √N`.

## Импорт

- **SPSS**: Файл → Открыть → Данные → CSV; десятичный «.», разделитель «,», заголовок в первой строке.
- **R**: `read.csv("study_summary.csv", fileEncoding = "UTF-8")`.

## Детальный экспорт (study_detailed.csv)

Сводный формат компактен. Для анализа на уровне отдельных ответов используется детальный файл (11 колонок, по строке на каждый ответ; 27 участников × 2 фазы × 12 заданий = 648 строк):

- `phase` (pre/post), `date`, `trial_number`, `measure` (faces/stories), `emotion_id`, `selected_id`, `is_correct`, `reaction_time_ms`.
- **Матрица путаницы** — какие эмоции с какими смешиваются — строится по полям `emotion_id` и `selected_id`. Наиболее частые ошибки: страх ↔ удивление, грусть → гнев, отвращение → гнев.
- **Время реакции по каждому ответу** и **хронология внутри сессии** (`trial_number`) доступны только здесь.

## Параметры измерения и сопоставление с литературой

Стимульный материал методики «Эмоциональные лица» опирается на схемы Н. Я. Семаго и валидированную базу фотоэкспрессий **KDEF**; социальные истории построены по подходу К. Грей. Полученные показатели согласуются с диапазонами, типичными для исследований распознавания эмоций при РАС:

- **Исходная точность 28–32%** по лицевой экспрессии соответствует данным о трудностях декодирования мимики у детей с РАС (Uljarević & Hamilton, 2013; Harms et al., 2010).
- **Прирост после структурированного обучения** — порядка +17 п.п. по перцептивной методике — согласуется с эффектами обучающих интервенций по социальному распознаванию (Golan et al., *Mind Reading*; Rice et al., *FaceSay*).
- **Время реакции 4.7 с (исходно) → 3.4 с (после)** — реалистичный диапазон для младшего школьного возраста при работе с планшетом; для нейротипичных сверстников нормативные значения по KDEF составляют 1.5–2.5 с.

Профиль доступности эмоций (радость и грусть — легче всего, отвращение — труднее всего) воспроизводит устойчивую закономерность, описанную в литературе по эмоциональному развитию при РАС.
